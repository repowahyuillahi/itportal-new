# Product Vision

## Tujuan

ITPortal menjadi tempat kerja internal tim IT untuk mencatat support harian dan
menghasilkan Support Maintenance Report bulanan tanpa menyusun tabel manual.

## Pengguna

- Admin IT: mengelola user, dealer, item, semua ticket, report.
- IT Staff: input dan update ticket.
- Manager: melihat dashboard dan export report.
- Viewer: read-only.

Dealer belum login sendiri di V1.

## MVP

- Login/logout.
- Dashboard bulanan.
- Input ticket.
- Kelola ticket.
- Master dealer.
- Master item/kategori.
- Filter report.
- Export Excel/PDF.
- Mobile-friendly ticket input.

## Future

- Export PPT.
- Notifikasi.
- SLA.
- Dealer self-service.
- Monitoring otomatis.
- Asset inventory.

## Sumber Istilah

`0426-WahyuIllahi.pptx` dan file Excel/CSV lain di root project berisi topik
meeting dan calon data import seperti server, CCTV, jaringan, backup, YDT,
DpackWeb, radio, Zahir, sertifikat, dealer, dan Support Maintenance Report.
Untuk V1, data tersebut harus diaudit dan di-map dulu, bukan di-hardcode.
